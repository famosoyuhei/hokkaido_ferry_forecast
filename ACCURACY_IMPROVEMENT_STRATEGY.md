## MARITIME_RESEARCH.md に基づく追加知見（2026-05-04）

MARITIME_RESEARCH.md のリサーチ結果と現行実装を照合し、以下の改善課題を特定した。

### 特定されたギャップ

#### ① 波高閾値が冬季実態に対して不足
- **現行**: 波高 2.0m以上で初めてスコアが入る（+15点）
- **実態**: 冬季（1〜3月）の平均有義波高は 2.5〜3.5m で、3.0〜3.5m 台でも欠航多発
- **影響**: 冬季の 2.5m 波高単体では LOW 判定になり見逃しやすい
- **対応方針**: 冬季データ（2026年12月〜2027年3月）が蓄積されてから実績ベースで再評価する

#### ② 礼文島（香深）と利尻島（鴛泊）の気象差が未反映
- **現行**: 出発港の実測気象を使うが、稚内→香深ルートでは航路中間が最も外洋性が高い
- **実態**: 礼文島は遮蔽物がなく、同条件でも鴛泊より風・波が強い傾向（MARITIME_RESEARCH §4）
- **影響**: 稚内→香深ルートのリスクが若干過小評価される可能性
- **対応方針**: Phase 3 で「航路別気象バイアス補正」として検討

#### ③ 4月定期ドック（気象外欠航）が精度計算を歪めている
- **現行**: 4/10〜11 の全便欠航が False Negative（精度0%）として計上される
- **実態**: ハートランドフェリーは毎年春（4月上旬〜中旬）に定期整備ドックを実施する模様
- **影響**: 4月上旬の精度が恒常的に低く見える
- **対応方針**: 毎年4月5〜15日前後を「計画外欠航除外期間」としてフラグ管理することを Phase 3 で検討

#### ④ ERA5 波高の離島近傍バイアス
- **現行**: Marine Archive API（ERA5ベース）から波高を取得
- **実態**: ERA5 の 31km 格子は利尻島・礼文島の島陰効果を表現できず、島近傍で波高を**過大推定**する可能性（MARITIME_RESEARCH §8）
- **影響**: 鴛泊・沓形・香深の波高が実際より高く計算されている可能性 → HIGH 判定が増える方向（False Positive を生みやすい）
- **対応方針**: 既知バイアスとして記録。実測値との乖離が確認されれば補正係数を検討

#### ⑤ 霧の継続時間を考慮していない
- **現行**: 1時間単位の視程値をリスクスコアに加算
- **実態**: 夏季（6〜8月）の移流霧は 2〜3日間継続することがあり、単発 1 時間とは欠航リスクが異なる
- **影響**: 短時間の霧と長期霧を同等に扱っている
- **対応方針**: 「連続 N 時間以上の視程不良」への追加スコアを Phase 4 で検討

## 将来計画（Phase 3-5）

### Phase 3: 閾値最適化とデータ品質向上（2026年12月〜2027年3月）

**前提**: 冬季欠航データ（2026年12月〜2027年3月）の蓄積が必要

- **冬季波高閾値の再評価**: 1.5〜2.0m 帯への加点を実績ベースで判断
- **航路別気象バイアス補正**: 稚内〜香深ルートに礼文島側の気象を追加加重
- **計画外欠航フラグ**: 毎年4月上旬の定期ドック期間を識別してスコアリングから除外
- **動的閾値調整システム** (`auto_threshold_adjuster.py`): 月別・航路別の最適閾値を自動計算
- 目安: 6ヶ月以上の冬季データが揃ってから実施

### Phase 4: 機械学習による欠航確率予測（2027年以降）

- Random Forest / 勾配ブースティングによる欠航確率モデル
- 入力特徴量候補: 風速・波高・視程・季節・ルート・直前 N 時間の気象トレンド
- 霧継続時間の特徴量追加
- 目安: 1年以上（冬季2シーズン）のデータ蓄積後

### Phase 5: フィードバックループ・可視化（2027年以降）

- 日次・週次・月次精度レポートの自動生成
- 精度トレンドグラフ（ダッシュボードへの組み込み）
- 異常検知システム（精度急低下時のアラート）
- GitHub Pages へのレポート自動デプロイ

## 教訓

- **誤ったデータで閾値調整しても意味がない**（季節調整を導入したが、欠航データのバグが原因だったため逆効果だった）
- まず正しいデータを確保してから閾値をチューニングすること
- 気象外欠航（ダイヤ切替・機材不具合等）は気象モデルで予測不可能。データ蓄積で季節パターンとして把握する
- **ERA5 は便利だが離島近傍の波高過大推定という既知バイアスがある**（MARITIME_RESEARCH §8 参照）
- 冬季（12〜3月）に欠航の 1/4〜1/3 が集中する。この期間のデータなしに閾値を語れない
