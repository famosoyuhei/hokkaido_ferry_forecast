## 1. 海域の地理的特徴

| 地点 | 緯度/経度 | 特記事項 |
|------|---------|---------|
| 稚内港 | N45.4° / E141.7° | 日本最北端の主要港 |
| 鴛泊港（利尻島東） | N45.2° / E141.2° | 利尻富士（1,721m）東麓 |
| 沓形港（利尻島西） | N45.4° / E141.1° | 利尻富士西側、荒天時の代替港 |
| 香深港（礼文島南） | N45.3° / E141.0° | 礼文島南部 |
| 稚内〜鴛泊 | 約 40 km | 日本海（宗谷海峡西口付近） |
| 稚内〜香深 | 約 55 km | 日本海（礼文水道） |

**重要**: フェリー航路は宗谷海峡（東側）を通らず、稚内**西方の日本海**を通る。宗谷海峡は稚内の東方でサハリンとの間（最狭部 約43km）にある別の海域。

## 2. 卓越気象パターン（季節別）

### 冬季（12〜3月）
- **卓越風向**: 北西〜西北西（シベリア高気圧からの寒気吹き出し）
- **風速**: 平均 15〜25 m/s、強風時 30〜40 m/s 超
- **特徴**:
  - 冬型気圧配置（西高東低）が週3〜4回のペースで持続
  - 日本海北部は吹送距離が長く、波浪が発達しやすい
  - 礼文島は遮蔽物がほとんどなく、利尻島より風が強い傾向

### 春季（4〜6月）
- **卓越風向**: 南〜南西（移行期）
- **特徴**:
  - 3月下旬〜4月は突発的な強風・高波が残る（「春の嵐」）
  - 低気圧通過時に南西→北西への急転換が起こりやすい
  - 5〜6月は**霧の最盛期**（後述）

### 夏季（7〜8月）
- **卓越風向**: 南東〜南（太平洋高気圧の周辺流）
- **特徴**:
  - 比較的穏やか（年間で欠航率が最低の時期）
  - 霧が多い（移流霧 → §5参照）
  - 台風接近時は急変するが北海道直撃はまれ

### 秋季（9〜11月）
- **卓越風向**: 北〜北東（徐々に冬型へ移行）
- **特徴**:
  - 10月以降、冬型気圧配置が急増
  - 台風の影響は9月に集中
  - 11月は欠航率が急上昇（冬への移行期）

## 3. 主要気象擾乱とフェリーへの影響

| 擾乱パターン | 頻度 | 風向 | 波高 | 継続時間 | 主な発生季節 |
|------------|------|------|------|---------|-----------|
| **冬型気圧配置** | 週3〜4回 | 北西〜西北西 | 3〜6m+ | 1〜5日 | 12〜3月（欠航最多） |
| **発達低気圧（爆弾低気圧）** | 月1〜3回 | 南→北西（急転換） | 急上昇（2m→5m+） | 1〜2日 | 10〜4月 |
| **南岸低気圧** | 月1〜2回 | 南〜南西 | 比較的低め | 0.5〜1日 | 1〜3月 |
| **台風** | 年1〜2回 | 変化大 | 大波 | 1〜3日 | 8〜9月 |

**欠航への影響が最も大きいのは冬型気圧配置の持続**。複数日にわたり連日欠航が続くケースはほぼこのパターン。

## 4. 局地気象現象

### 利尻富士（1,721m）の地形効果

| 現象 | 発生条件 | 影響箇所 | フェリーへの影響 |
|------|---------|---------|--------------|
| 山岳波 | 強い西〜北西風 | 風下（東・南東側）に波状雲・乱気流 | 鴛泊港周辺の突風 |
| フェーン現象 | 南西風 | 鴛泊港（東側）で気温上昇・視程改善 | 西岸（沓形）は荒天 |
| Gap Wind（局地強風） | 北西風が山体北側を回り込む | 鴛泊港付近 | 局所的な突風リスク |

**稚内〜鴛泊航路は利尻島の北東〜東を通るため、利尻富士の遮蔽効果が部分的にある。**

### 礼文島の特徴
- 最高点 約490m（澄海岬付近）で山岳効果は限定的
- 日本海から吹く北西風をほぼそのまま受ける
- 稚内〜香深航路の後半（礼文島側）は外洋性の波浪にさらされる
- **同じ気象条件でも利尻島より礼文島側の方が波・風が厳しい傾向**

## 5. 霧・視程不良のパターン

| 月 | 発生頻度 | 霧の種類 | 主な原因 |
|----|--------|---------|---------|
| 1〜3月 | 低 | 放射霧 | 晴天夜間の放射冷却（陸上） |
| 4〜5月 | 中（増加） | 移流霧 | 暖かい南風 + 冷たい海面 |
| **6〜8月** | **最高** | **移流霧（卓越）** | 暖湿な南〜東南東風 + 親潮の低海面水温 |
| 9〜11月 | 中 | 蒸気霧 | 海面 > 大気気温（秋季） |
| 12月 | 低〜中 | 地吹雪 | ブリザード時に視程がほぼゼロになる場合も |

**移流霧（6〜8月）**: 日中でも解消しない場合がある。南東風が持続すると2〜3日継続することも。

### 視程とフェリーへの影響（業界標準）
| 視程 | 対応 |
|------|------|
| 3km 以上 | 通常運航 |
| 1〜3km | 速度低下・慎重航行 |
| 0.5〜1km | 欠航検討 |
| 0.5km 未満 | 原則欠航または待機 |

**現行モデルの視程閾値（< 1.0km で +20点、< 3.0km で +10点）は業界標準と概ね整合。**

## 6. 波高の特徴

### 宗谷海峡（東側）vs. フェリー航路（日本海西側）の違い

| 特徴 | 宗谷海峡（東側）| フェリー航路（日本海西側）|
|------|--------------|----------------------|
| 吹送距離 | 短い（〜サハリン約43km） | 長い（〜朝鮮半島・ロシア沿岸 500km+） |
| 冬季波高 | 比較的低め（2〜3m） | 高い（4〜6m 以上になりやすい） |
| 卓越波向 | 北〜北北東 | 北西〜西北西 |
| うねり | 少ない | 遠方発生のうねり（周期長い）が入る |

### 月別波高目安（稚内西方・日本海）

| 月 | 平均有義波高 | 嵐時の最大 |
|----|------------|---------|
| 1〜2月 | 2.5〜3.5m | **6〜8m** |
| 3月 | 2.0〜3.0m | 5〜7m |
| 4月 | 1.5〜2.5m | 4〜6m |
| 5月 | 1.0〜2.0m | 3〜5m |
| 6〜8月 | 0.7〜1.5m | 2〜3m |
| 9月 | 1.0〜1.8m | 3〜5m |
| 10月 | 1.5〜2.5m | 4〜6m |
| 11〜12月 | 2.0〜3.5m | 5〜8m |

**→ 現行モデルの波高閾値（2.0m: +15点、3.0m: +30点、4.0m: +40点）は正方向だが、冬季2〜3m台での欠航をカバーするには不足の可能性。**

## 7. 海流・潮汐

- **対馬暖流**: 日本海を北上。稚内沖の冬季海面水温を約2〜6℃に保つ。暖流と冷気の接触が降雪・霧の源。
- **宗谷暖流**: 対馬暖流の支流が宗谷海峡を経由してオホーツク海へ流入（北向き最大0.5〜1.0kt）。
- **潮位差**: 稚内周辺は最大50〜80cm（日本海側、小さめ）。フェリー運航への直接影響は限定的。
- **流氷**: オホーツク海側に発生するが、日本海側のフェリー航路には通常到達しない。

## 8. 気象予報技術と精度

### JMA MSM（メソスケールモデル）

| 項目 | 仕様 |
|------|------|
| 水平解像度 | **5km格子** |
| 予報時間 | **39時間先**（1時間ステップ） |
| 更新頻度 | **1日8回**（3時間おき） |
| データ同化 | 4次元変分法（4D-Var） |

**制約事項**:
- 5km格子では利尻富士（1,721m）の地形効果を十分表現できない
- 北海道北部は観測点が少なく（AMeDAS・気象レーダー）、予報精度がやや劣る
- 海上は陸上より観測データが少なく初期値精度に制約

### 気象庁 沿岸波浪モデル

| 項目 | 仕様 |
|------|------|
| 日本沿岸解像度 | 約5〜10km |
| 予報時間 | 84時間先 |
| 更新頻度 | 1日4回 |
| 出力 | 有義波高・波周期・波向（風波＋うねり分離） |

**精度の注意点**:
- 外洋域（有義波高誤差 平均0.3〜0.5m）は比較的良好
- 島嶼・地形遮蔽のある沿岸域では誤差が増大
- 利尻島・礼文島周辺では波高**過大推定**のリスク（島陰効果を格子で表現しきれない）

### 稚内地方気象台の警報・注意報基準（目安）

| 種別 | 発表基準（目安） |
|------|--------------|
| 暴風警報 | 最大風速 25 m/s 以上（海上 20 m/s 以上） |
| 暴風雪警報 | 暴風 + 降雪・吹雪の組み合わせ |
| 波浪警報 | 有義波高 **4m 以上**（稚内沖） |
| 波浪注意報 | 有義波高 2.5〜3.0m |
| 濃霧注意報 | 視程 200m 以下（陸上）/ 500m 以下（海上） |
| 特別警報（暴風） | 最大風速 おおむね 35 m/s 以上 |

**→ 波浪警報（4m）の発表とフェリー欠航判断はほぼ連動すると推定される。**

### Open-Meteo ERA5 再解析データの精度

| 気象要素 | 評価 | 注意点 |
|---------|------|--------|
| 風速（10m高度） | 中〜良好 | 31km格子のため利尻富士の地形効果は表現不可 |
| 波高 | **要注意** | 島嶼近傍で誤差大（島陰効果を過大評価） |
| 視程 | 悪〜中 | 直接変数なし。相対湿度から推定が必要 |
| 気温 | 良好 | |
| 降水 | 中 | 山岳性降水を過小評価（利尻富士） |

**ERA5の格子解像度**: 31km（水平）。利尻島（直径約20km）と礼文島（幅約8km）は**1〜2格子以下の大きさ**のため、島の地形効果はほぼ表現されない。

## 10. 欠航判断基準（調査・推定）

### 気象閾値（推定）

| 気象要素 | 欠航検討 | 原則欠航 |
|---------|---------|---------|
| 風速（海上） | 20〜25 m/s | **25〜30 m/s 以上** |
| 有義波高 | 3.0〜3.5m | **4.0m 以上** |
| 視程（海上） | 1.0km 未満 | 0.5km 未満 |
| 波浪警報 | 状況による | 多くの場合は欠航 |

### 季節差（推定）
- **冬季（12〜3月）**: 閾値がやや低め（塩害・強風時の停泊リスク・積雪による甲板凍結も考慮）
- **夏季（7〜8月）**: 閾値がやや高め

### 月別推定欠航率

| 月 | 推定欠航率 | 主要原因 |
|----|----------|---------|
| **1〜2月** | **25〜35%** | 冬型強風・高波（最悪期） |
| 3月 | 20〜30% | 冬型弱まるが低気圧多い |
| 4月 | 15〜25% | 春の嵐・ダイヤ切り替え |
| 5月 | 5〜10% | 穏やか |
| 6〜7月 | 5〜10% | 霧による視程不良が主 |
| **7〜8月** | **2〜5%** | **年間最低（欠航最少期）** |
| 9月 | 5〜15% | 台風・秋低気圧 |
| 10月 | 15〜25% | 冬型増加 |
| **11〜12月** | **20〜35%** | 冬型卓越・高波 |

**→ 冬季（1〜3月）は便の1/4〜1/3が欠航する想定。この期間のデータ蓄積が予報モデルの精度評価に最も重要。**

## 11. 予報システムへの示唆

### 現行モデルと照合した改善候補

| 発見事項 | 現行の対応 | 推奨アクション |
|---------|-----------|-------------|
| 冬季は波高2.0〜3.0m台でも欠航する | 波高2.0m以上で+15点（不足の可能性） | 冬季データ蓄積後に閾値を再評価 |
| 礼文島（香深）は利尻島（鴛泊）より風・波が厳しい | 同一気象データで計算 | 将来的に航路別気象マッチングを強化 |
| 利尻富士の地形効果でGap Windが発生 | ERA5の31km格子では表現不可 | 稚内気象台・AMeDASデータとの補完を検討 |
| 霧（6〜8月）は複数日継続することがある | 1時間単位の視程データ使用 | 継続時間の考慮（将来フェーズ） |
| 港内波浪が欠航判断に影響（防波堤越波） | 未実装 | データが取得できれば追加 |
| 4月上旬の定期ドック（気象外欠航） | 欠航データとして計上 | 毎年4月上旬前後を「気象外期間」として除外フラグを検討 |
| ERA5波高は島嶼近傍で過大推定 | Marine APIを使用中 | 現行のまま継続（問題認識として保持） |

### データ蓄積の優先度

1. **最優先（今年12月〜来年3月）**: 冬季の欠航実績データ。閾値最適化に不可欠。
2. **優先（今年6〜8月）**: 霧による視程不良と欠航の関係。夏季の精度向上に寄与。
3. **中期（2年間）**: 月別欠航率の実測値で上記の推定値を検証。

### ERA5 vs. JMA の使い分け方針

| 用途 | 推奨データ | 理由 |
|------|----------|------|
| Hindcast精度計算（過去） | ERA5（Open-Meteo Archive） | 1940年〜取得可能、無料、1時間解像度 |
| 7日先予報リスク | JMA MSM + Open-Meteo Forecast | リアルタイム予報精度 |
| 波高（予報） | Open-Meteo Marine API | 海象予報に特化 |
| 波高（過去・検証） | ERA5波浪変数 | ただし島嶼近傍の過大推定に注意 |
