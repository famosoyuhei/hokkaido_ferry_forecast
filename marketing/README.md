# marketing/ フォルダ 管理ガイド

**最終更新**: 2026-05-03  
**バージョン**: 1.0

このフォルダは、フェリー運航予報アプリの**営業・広告アウトプット**を一元管理するためのものです。  
コピー・動画・印刷物・SNS素材など、制作物の種類ごとにサブフォルダに分けて保存します。

---

## フォルダ構造

```
marketing/
├── README.md          ← このファイル（フォルダの説明・更新ルール）
│
├── copy/              ← テキストコピー（完成版の文言）
│   ├── README.md
│   ├── flyer_business_v1.0.txt    業者向けチラシ本文
│   └── flyer_tourist_v1.0.txt     観光者向けチラシ本文
│
├── print/             ← 印刷物ファイル（チラシ・ポスターのPDF・画像）
│   └── README.md
│
├── video/             ← 動画関連（スクリプト・完成動画ファイル）
│   ├── README.md
│   └── script_30sec_v1.0.txt      30秒動画スクリプト（テキスト版）
│
├── sns/               ← SNS・LINE投稿用テキスト
│   ├── README.md
│   └── posts_v1.0.txt             投稿文一覧
│
└── assets/            ← 素材ファイル（アプリのスクリーンショット・写真など）
    └── README.md
```

---

## ファイル命名ルール

```
[種類]_[ターゲット略称]_v[バージョン].[拡張子]

例:
  flyer_business_v1.0.txt   → 業者向けチラシ テキスト 初版
  flyer_business_v1.1.txt   → 業者向けチラシ テキスト 軽微修正版
  flyer_business_v2.0.txt   → 業者向けチラシ テキスト 大幅改訂版
  poster_a3_v1.0.pdf        → A3ポスター 初版
  video_30sec_v1.0.mp4      → 30秒動画 完成版
```

**ターゲット略称**:
- `business` = 飲食・小売業者向け
- `tourist` = 観光者・リピーター向け
- `both` = 両ターゲット共用

**バージョン番号の使い分け**:
- `v1.0 → v1.1`：文言の微調整・誤字修正など小さな変更
- `v1.0 → v2.0`：ターゲット・メッセージ・構成を大きく変えた場合

## 更新のしかた

1. **既存ファイルは上書きせず、新バージョンとして別名で保存する**  
   例：`flyer_business_v1.0.txt` を直したいとき → `flyer_business_v1.1.txt` を新規作成

2. **古いバージョンはそのまま残す**（比較・差し戻しができるように）

3. **このREADMEの「更新履歴」テーブルに1行追記する**
