## 完成動画ファイルの保存について

完成した動画ファイル（.mp4 など）はサイズが大きいため Git 管理には向きません。  
Google ドライブや外部ストレージに保存し、このREADMEにリンクを記載する運用を推奨します。

例：
```
| video_30sec_v1.0.mp4 | 30秒完成動画 | Google Drive: [リンクをここに記載] |
```

## 更新ルール

- スクリプト修正は新バージョンとして別名保存（script_30sec_v1.1.txt など）
- このREADMEのファイル一覧テーブルに追記する
