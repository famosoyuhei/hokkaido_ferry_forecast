# print/ — 印刷物ファイル（チラシ・ポスター）

デザインが完成したチラシ・ポスターの PDF・画像ファイルを保存するフォルダです。

## ファイル一覧

| ファイル | 内容 | 更新日 |
|---------|------|--------|
| （制作後に追加） | | |

## ファイル命名例

```
flyer_business_a4_v1.0.pdf     業者向けチラシ A4 初版
flyer_tourist_a4_v1.0.pdf      観光者向けチラシ A4 初版
poster_both_a3_v1.0.pdf        両ターゲット共用ポスター A3 初版
```

## 更新ルール

- デザインを修正したら新バージョンとして別名保存（v1.1, v2.0 など）
- 古いバージョンはそのまま残す（印刷物は差し戻しが必要になることがあるため）
- このREADMEのファイル一覧テーブルに追記する

## テキスト素材の参照先

チラシ・ポスターに使う文言は [../copy/](../copy/) フォルダの対応バージョンを使用してください。
