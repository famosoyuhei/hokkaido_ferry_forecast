# assets/ — 素材ファイル

広告制作に使う素材（アプリのスクリーンショット・写真・ロゴなど）を保存するフォルダです。

## ファイル一覧

| ファイル | 内容 | 更新日 |
|---------|------|--------|
| （素材追加時に記載） | | |

## 保存する素材の例

- アプリのスクリーンショット（7日間予報画面、リスクレベル表示など）
- 撮影した写真（港・食堂・観光者の様子など）
- ロゴ・アイコン画像

## ファイル命名例

```
screenshot_forecast_7days.png    7日間予報画面のスクリーンショット
screenshot_risk_high.png         HIGH表示のスクリーンショット
photo_port_oshidomari.jpg        鴛泊港の写真
```

## 注意

- 人物が写っている写真は、被写体の使用許可を取ってから保存・使用すること
- サイズが大きいファイル（10MB超）は Google ドライブ等に保存し、このREADMEにリンクを記載する
