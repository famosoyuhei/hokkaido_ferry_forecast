# sns/ — SNS・LINE投稿用テキスト

X（旧Twitter）・Instagram・LINEグループへの投稿文を保存するフォルダです。

## ファイル一覧

| ファイル | 内容 | 更新日 |
|---------|------|--------|
| posts_v1.0.txt | 業者向け・観光者向け投稿文一覧（初版） | 2026-05-03 |

## 更新ルール

- 投稿文を追加・修正したら新バージョンとして別名保存（posts_v1.1.txt など）
- このREADMEのファイル一覧テーブルに追記する
- 実際に投稿して反応がよかった文言には「★効果あり」とコメントを添える
