# 営業・広告素材集：利尻島・礼文島 飲食・小売業者／島と本土を行き来する観光者向け

**最終更新**: 2026-05-03  
**バージョン**: 1.0  
**対象ターゲット**:  
① 利尻島・礼文島の飲食店・小売店（食材・商品の仕入れをフェリーに依存）  
② 島と本土を頻繁に行き来するリピーター観光者・島出身者

---

## ターゲット分析

### ターゲット①：飲食・小売業者

| 項目 | 内容 |
|------|------|
| 具体例 | 飲食店（民宿・食堂・カフェ）、食料品店、土産物店、ガソリンスタンドなど |
| フェリーとの関係 | 食材・飲料・日用品・商品の仕入れがすべてフェリー経由 |
| 欠航による被害 | 食材が届かずメニュー変更を強いられる／在庫切れで販売機会を失う／発注タイミングを誤ると欠品が長引く |
| 響くキーワード | 仕入れ計画・在庫切れゼロ・食材ロスを減らす・利益・安心して発注できる |
| スマホ習熟度 | 中。飲食・小売系SNS（Instagram等）を使う方も増えている |

### ターゲット②：リピーター観光者・島出身者

| 項目 | 内容 |
|------|------|
| 具体例 | 利尻・礼文をリピートする旅行者、島外に出た島出身者（帰省）、写真家・ライター・ダイバーなど |
| フェリーとの関係 | 旅程全体がフェリーの運航に左右される。欠航で予定が1〜2日単位で崩れる |
| 欠航による被害 | 宿・交通のキャンセル費用発生／帰りのフライトに間に合わない／目的の体験ができない |
| 響くキーワード | 旅程を守る・安心して予約できる・欠航で足止めされない・事前準備 |
| スマホ習熟度 | 高め。旅行アプリやSNSに慣れている層が多い |

### 共通インサイト

> **業者向け**: 「欠航の翌日、冷蔵庫が空になって初めて後悔する。でも3日前にわかってれば動けた。」  
> **観光者向け**: 「欠航で足止めされた経験が一度でもあると、次は必ず事前に調べるようになる。」

**見出し（大）**


**見出し（中）**
```

7日先まで毎朝確認できる無料アプリ
```

**本文**
```
島の飲食店・小売店にとって、フェリーは「動脈」です。

欠航が重なると食材が届かず、
メニューを変えざるを得ない。
在庫が切れて、お客様に申し訳ない。
そんな経験、ありませんか？

このアプリは、稚内⇔利尻・礼文間のフェリーについて
風速・波高・視程のデータをもとに
今後7日間の欠航リスクを毎朝更新します。

● 赤（HIGH）   欠航の可能性が高い日
● 橙（MEDIUM） 要注意の日
● 緑（MINIMAL）安定している日

3〜4日先に赤が見えたら、早めに発注する。
緑が続くなら、通常通りの計画でOK。

無料・登録不要。スマホで今すぐ使えます。
```

**URL（大きく）**
```
https://web-production-a628.up.railway.app/
```

**注記**
```
※気象データをもとにした予測です。
　運航可否の最終確認はハートランドフェリー公式サイトをご確認ください。
```

## 30秒広告動画スクリプト

### 制作方針

- **長さ**: 30秒
- **想定媒体**: YouTube広告 / Instagram Reels / 観光施設のモニター
- **トーン**: 落ち着いていて誠実。島の美しさを背景に、「役に立つ」を前面に
- **字幕**: 全セリフに字幕（音声オフでも伝わるように）
- **2ターゲット共用**: 前半を業者、後半を観光者で構成し、両方に届くつくりにする

---

### シーン構成（業者×観光者 共用版）

```
00:00〜00:06  【オープニング：島の朝】
映像: 夜明けの鴛泊港。フェリーが岸壁に着いている。波の音。
     港近くの食堂に明かりがつく。
テロップ: 「利尻島・礼文島の仕事と旅は、フェリーでつながっている。」
ナレーション: （なし。映像と波音のみ）

---

00:06〜00:13  【問題提示】
映像: 食堂の女将（50代）が届いた食材を確認している。
     電話が鳴る。「今日、欠航です」。
     女将が冷蔵庫を開けて、食材の少なさに目を細める。
テロップ: 「欠航を知ったのは、当日の朝だった。」
ナレーション: 「欠航がわかるのは、いつも当日の朝。」

---

00:13〜00:20  【解決策の提示】
映像: 同じ女将が翌朝、スマホでアプリを開く。
     画面に「3日後：HIGH（赤）」が表示される。
     女将がうなずきながら発注書を書き始める。
テロップ: 「7日先まで、欠航リスクがわかる。」
ナレーション: 「今は、7日先まで欠航リスクを確認できます。
              赤い日がわかれば、前日に多めに仕入れておける。」

---

00:20〜00:26  【観光者カット】
映像: スマホを持った旅行者（30代カップル）がフェリー乗り場にいる。
     アプリを開き、帰りの日が「MINIMAL（緑）」なのを確認して笑顔になる。
テロップ: 「帰りの便も、事前に確認できる。」
ナレーション: 「旅の計画を立てるときにも使えます。」

---

00:26〜00:30  【エンディング】
映像: アプリのトップ画面（スマホ実機）。利尻富士が背景に溶け込む。
テロップ（大）: 「フェリー運航予報　無料・登録不要」
テロップ（小）: https://web-production-a628.up.railway.app/
ナレーション: 「無料・登録不要。今日から使えます。」
```

### 撮影・制作メモ

| 項目 | 内容 |
|------|------|
| 推奨撮影時期 | 7〜8月（天候・明るさ・観光者の動きが揃う） |
| ロケ地候補 | 鴛泊港／島内の食堂・土産物店（許可取得）／礼文島香深港 |
| キャスト | 地元の飲食店主に出演依頼が理想。親近感と信頼性が増す |
| BGM | 穏やかなアコースティック系。波の環境音を活かす |
| 字幕フォント | 太めのゴシック体・白文字＋黒縁（高齢者・屋外視聴でも読みやすい） |
| スマホ画面 | iPhone の画面収録で実際のアプリ画面を録画して使用 |
| 尺のバリエーション | 15秒版（業者のみ）・15秒版（観光者のみ）も別途制作推奨 |
