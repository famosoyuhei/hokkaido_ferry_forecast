# 営業・広告素材集：利尻島 漁師・昆布干し従事者向け

**最終更新**: 2026-05-03  
**バージョン**: 1.0  
**対象ターゲット**: 利尻島在住の漁師・昆布干し従事者

---

## ターゲット分析

### 漁師

| 項目 | 内容 |
|------|------|
| 年齢層 | 40〜70代が中心 |
| フェリーとの関係 | 漁獲物を稚内・本州へ出荷／漁具・資材の受け取り |
| 天候リスク | 欠航で出荷できず鮮度が落ちる／必要な資材が届かず出漁できない |
| スマホ習熟度 | 低〜中。LINEや天気アプリ程度は使う |
| 響くキーワード | 損しない・安心・シンプル・無料・仲間も使ってる |

### 昆布干し従事者

| 項目 | 内容 |
|------|------|
| 作業時期 | 主に7〜9月（利尻昆布の収穫・乾燥シーズン） |
| フェリーとの関係 | 乾燥昆布の本州出荷タイミング管理／買付業者の来島スケジュール調整 |
| 天候リスク | フェリー欠航 ≒ 強風・荒天 ≒ 乾燥作業も中断。二重の打撃 |
| 響くキーワード | 段取り・無駄にしない・先読み・昆布・天気 |

### 共通インサイト

> 「欠航になってから慌てても遅い。でも、前の日にわかってれば対処できる。」

この感覚を軸に訴求する。

## コピーライティング素材集

### A. チラシ・ポスター用（A4想定）

**見出し（大）**
```
明後日の欠航、もう知ってますか？
```

**見出し（中）**
```
フェリー欠航リスクを7日前から確認できる
無料スマホアプリができました
```

**本文**
```
利尻島から稚内へのフェリーが欠航すると、
出荷が遅れる、資材が届かない、
せっかくの昆布が出せない——

そんな「知らなかった損」を減らすために、
このアプリをつくりました。

毎朝6時以降に確認するだけ。
今日から7日間、欠航しそうかどうかが
スマホの画面に色でわかります。

● 赤（HIGH）    欠航の可能性が高い
● 橙（MEDIUM）  様子を見て判断を
● 緑（MINIMAL） ほぼ大丈夫

無料・登録不要。スマホがあれば今すぐ使えます。
```

**URL表示（大きめに）**
```
https://web-production-a628.up.railway.app/
```

**フッター補足**
```
※気象データをもとにした予測です。
　出航可否の最終確認はハートランドフェリー公式サイトでお願いします。
```

## 30秒広告動画スクリプト

### 前提条件
- 長さ: 30秒
- 想定媒体: YouTube広告 / SNS動画 / 地域の広報モニター
- トーン: 静かで誠実。煽らない。実際の仕事の現場感を大切に
- 字幕: 全セリフに字幕（音声なしで見ても伝わるように）

---

### シーン構成

```
00:00〜00:05  【オープニング】
映像: 夜明けの利尻島。港に漁船が並んでいる。波の音。
テロップ: （なし）
ナレーション: 「漁師の朝は、天気から始まる。」

---

00:05〜00:12  【問題提示】
映像: 漁師（50代男性）がスマホを見ながら眉をひそめている。
     ニュースかSNSで「欠航」の文字。
テロップ: 「欠航を、知ったのは当日の朝だった。」
ナレーション: 「でも、欠航を知るのが当日では——遅い。」

---

00:12〜00:21  【解決策の提示】
映像: 同じ漁師が、翌朝スマホでアプリを開いている。
     画面に「3日後：HIGH（赤）」が表示される。
     漁師がうなずき、電話をかけ始める。
テロップ: 「7日先まで、欠航リスクがわかる。」
ナレーション: 「今は、7日先まで欠航リスクがわかります。
              赤い日がわかれば、前もって動ける。」

---

00:21〜00:27  【昆布干し従事者へのカット】
映像: 海岸で昆布を干している女性（60代）。
     スマホ画面に「今週：MINIMAL（緑）」が並んでいる。
     にっこりして昆布を広げ始める。
テロップ: 「今週は安心して干せる。」
ナレーション: 「荒れる日も、穏やかな日も、先に知っておける。」

---

00:27〜00:30  【エンディング】
映像: アプリのトップ画面（スマホ実機画面）
テロップ（大）: 「フェリー運航予報　無料・登録不要」
テロップ（小）: https://web-production-a628.up.railway.app/
ナレーション: 「無料・登録不要。今日から使えます。」
```

### 撮影・制作メモ

| 項目 | 内容 |
|------|------|
| 推奨撮影時間帯 | 早朝5〜7時（実際の漁師の朝と合わせる） |
